フルスタックのSaaS型Webアプリを構築してください。
このアプリは「営業リスト収集・AIメール生成・メール配信」を行う営業支援ツールです。

# 🧠 コンセプト
白黒でミニマルかつスタイリッシュなUIの営業自動化ダッシュボード。

機能は以下：
1. 検索APIで企業サイトを取得
2. フィルタ＆スコアリングで精度の高いリストを生成
3. クローラーで会社情報・問い合わせ先を抽出
4. OpenAIで営業メールを自動生成
5. SMTPでメール送信
6. ステータス・履歴管理

# 🧩 技術スタック

フロントエンド:
- Next.js 14（App Router）
- TypeScript
- Tailwind CSS（白黒ミニマルデザイン）
- shadcn/ui

バックエンド:
- Next.js API Routes または Route Handlers
- Prisma ORM
- Neon PostgreSQL

認証:
- NextAuth（メール＆パスワード）

メール送信:
- Nodemailer（Hostinger SMTP使用）

AI:
- OpenAI API（gpt-4o-mini または最新モデル）

クローラー:
- Axios + Cheerio（高速スクレイピング）
- Playwright（JSレンダリングページ対応）

検索:
- Brave Search API（メイン）
- 将来的にGoogle APIに差し替え可能な構造にする

キュー:
- cron + 非同期処理（Redis不要の軽量構成）

# 🔍 検索ロジック

事業ごとに検索クエリを生成する：

例：
"会社概要" 軽貨物 東京
"お問い合わせ" 美容クリニック 渋谷
"会社情報" 建設会社 神奈川

以下の除外を必ず行う：
- 求人サイト
- ランキングサイト
- 比較サイト
- 大手ポータル

# 🧠 スコアリングロジック

各サイトにスコアをつける：

+3 /contact がある
+3 /company or /about がある
+3 独自ドメイン
+2 .co.jp または .jp
-5 求人サイト
-5 比較サイト

5点以上のみ保存

# 🕷 クローラー

各サイトから以下を取得：
- メール（mailto含む）
- 問い合わせページURL
- 会社名
- 電話番号
- 住所

優先探索：
/contact
/company
/about
/privacy

# 🤖 AIメール生成（OpenAI）

以下の変数を使って営業メールを生成：

- company_name
- service_name
- service_url

自然で営業感の強い文章を生成すること。

# 📧 メール送信

- Nodemailerを使用
- 1通ずつ送信（スパム対策）
- 送信間隔を設ける（例：10秒）
- 配信停止リンクを必ず挿入
